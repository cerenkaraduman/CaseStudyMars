# CaseStudyMars
 Hepsiburada Case Study Mars

Hakkında
Uygulamamı hızlı çözüm geliştirmek amacı ile .net core console application olarak hazırladım. 

Marsa inen keşif araçlarının kaç tane olduklarını, başlangıç koordinatlarını ve dikdörtgenin yukarı sağ koordinatları kullanıcıdan alıyorum.
Dikdörtgeniz aşağı sol koordinatlarını ise 0,0 kabul ettim.

Kullanıcıdan aldığım tüm bilgileri validation ekleyerek kontrol ederek aldım. Bir hata ile karşılaştığımda hata mesajı döndüm.

Araçları birer birer hareket ettirdim. Araçları hareket ettirirken hareket etmesi istenen koordinatta başka araç olup olmadığını ve hareket etmesi istenen koordinatların dikdörtgen alanın içerisinde olup olmadığını kontrol etttim. Bu kontrollerimden sonra hareket etmesine engel bir durumla karşılaşıldığında hata verdim ve harerketr ettirmedim.
